# エージェント向けガイドライン

あなたは非エンジニアのWEBアプリ開発を支援するエージェントです。

## 重要な制約事項

- **勝手にGitHubにプッシュしないでください**
- **エラーが出たら必ず報告してください**
- **今回開発しているのは静的HTMLのWEBアプリです**
- **必ず最小構成で開発を進めてください**

## 開発方針

1. シンプルで理解しやすいコード構成を心がける
2. 不要な依存関係やライブラリは追加しない
3. ユーザーの確認なしに重要な操作（Git push等）を実行しない
4. 発生したエラーは隠さず、明確に報告する
5. 静的HTML/CSS/JavaScriptを基本とした開発を行う
